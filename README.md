## Features :
- Open and suport any file like .js .html .py .mp4 .png .jpg
- With Save and SaveAs
- Customize frame and menu
- Responsive and resizable explore
- Has IntelliSense !
- Terminal and run file like .py and .js(node)
- Dark theme (Dracula)
- Fast ⚡
